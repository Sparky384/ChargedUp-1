SYSID Setup Process:
-Set Generator Information
-Reset Telemetry
-Put In Logger Info (Client NT4, Units per rotation 0.0473, 5810 port (default), sysid (network Identity),
-hit "apply" under logger.
-run tests
-enable in driverstation

Getting Data Process:
-Under "Logger" section: choose a save location for the data file
-Save the data file
-Under "Analyzer" section: choose the saved file
